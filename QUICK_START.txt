## 📊 Để Hiểu Cách Nó Hoạt Động

### Forward Diffusion (Thêm Nhiễu)
```
x₀ (data gốc)
   ↓ add noise
x₁ (ít noise)
   ↓ add noise
x₂ 
   ...
x_T (pure noise)

Công thức: x_t = √ᾱ_t·x₀ + √(1-ᾱ_t)·ε
```

### Model Learning
```
Input: (x_t, t)
   ↓ UNet
Output: ε_pred (dự đoán noise)

Loss = ||ε_pred - ε||²  (so sánh với noise thật)
```

### Reverse Process (Bỏ Nhiễu)
```
x_T (pure noise)
   ↓ model dự đoán & bỏ noise
x_{T-1} (ít noise hơn)
   ↓ lặp lại
x_0 (data sinh ra)
```

## 🎯 Các Biến Chuyển Chính

| Biến | Vị Trí | Ảnh Hưởng | Mục Đích |
|------|--------|----------|---------|
| `beta_schedule` | GaussianDiffusion1D init | Kiểm soát noise progression | Thay 'linear'/'cosine'/'sigmoid' |
| `timesteps` | GaussianDiffusion1D init | Độ chi tiết của process | Tăng → tốt hơn, chậm hơn |
| `noise` (q_sample) | GaussianDiffusion1D.q_sample() | Loại noise phân phối | Gaussian/Uniform/Laplace |
| `STRENGTH` (q_sample) | GaussianDiffusion1D.q_sample() | Mức độ noise mạnh | 0.5/1.0/1.5 |
| `num_epochs` | Training loop | Tổng iterations | Tăng → tốt hơn, chậm hơn |
| `batch_size` | Training | Batch size | 32/64/128 |
| `learning_rate` | Optimizer | Learning rate | 1e-3/1e-4/1e-5 |
